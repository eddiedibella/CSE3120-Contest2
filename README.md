# CSE3120-Contest2
contest repository for CSE3120 game (second contest). Group members Eddie Di Bella, Jacob Ewasko

## Compile/Run Instructions
- Go to your current Visual Studio 2022 32 bit project
- Right Click the project -> Add -> Existing Item -> (Then search for the .asm file from this repository)
- Build Solution
- Debug -> Start Debugging to run the program

## GAMEPLAY INSTRUCTIONS
- Use W A S D to navigate the player around the map
- Use R to drink water, use F to eat food, use M to use medicine
- Use q to quit
